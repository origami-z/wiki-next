Added full internationalization for game data:

```json

    "name": "烬炎骑士",

  }
}
```

**Updated Pages** to pass locale to loaders:

Now when users visit `/zh/games/wittle-defender/heroes`, they'll see Chinese translations for hero names and descriptions.
